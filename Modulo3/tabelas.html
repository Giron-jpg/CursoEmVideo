Criando Tabelas

Utilizamos tabelas em html para apresentação de dados, porém antigamente esses modelos já foram utilizados para construção de sites (não mais recomendado). Para criar uma tabela em HTML usamos a tag table e dentro dela devemos seguir uma hierarquia exemplificada a seguir.

        Hierarquia de Tabelas (simples)
        TABLE = tabela
            CAPTION = Título da tabela
            TABLE HEADER (tag thead)= Cabeçalho da tabela
            TABLE BODY (tag tbody)= corpo da tabela
            TABLE FOOT (tag tfoot)= pé da tabela
            TABLE HEAD (tag th)= Título para a linha ou coluna da tabela
            TABLE ROW (tag tr)= Linha da tabela
            TABLE DATA (tag td)= Dados de tabela

Sendo assim usou-se as seguintes tags mencionadas para elaborar uma tabela 3x3

Para configurar seu formato usa-se comandos CSS como: border para atribuir linhas aos elementos; border-collapse que por padrão vem como separete, mas mudando para collapse todos os dados se agrupam, entre outros. É preciso levar em consideração qual parte da tabela deve ser formatada se são as linhas ou dados.

OBS.: Por padrão o th vem como negrito e centralizado

Importante: ao inserir um th é necessário acrescentar o parâmetro scope e informar no seu valor a direção em que se segue o conteúdo do título daquela coluna (ex.: scope="col" se o dado ficar na mesma coluna). Caso os dados ocupem varias colunas ou linhas usamos colgroup ou rowgroup

Efeito Zebrado

Para criar o efeito zebrado utilizamos a pseudo classe nth-child(2n) que permite a implementação de algum parâmetro em classes filhas de 2 em 2 elementos. O valor dentro do parênteses indica o padrão para as classes filhas podendo ser numérico, impar (odd ou 2n-1) ou par (even ou 2n). Nesse caso ele foi aplicado sobre as linhas (tr).

Mesclagem de Celula

Para mesclar celulas não é necessário comandos css, apenas acrescentar parâmetros a linha ou dado que se pretende mesclar. Podemos mesclar celulas na horizontal (pormeio do parametro colspan) e na vertical (através do comando rowspan)
