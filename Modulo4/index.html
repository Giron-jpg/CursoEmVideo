iFrames

A efeito de comparação são estruturas similares a janelas, permitindo visualizar conteúdos de outros sites. Um exemplo é o método de incorporar (compartilhar) vídeos do youtube. Por padrão esse modelo possui estrutura inline e um tamanho de 300x150px.

Podemos utilizar o iframe para exibir algum conteúdo previamente carregado na mesma página. Para isso precisamos atribuir um name ao iframe e em seguida inserir no link um target para esse nome.
