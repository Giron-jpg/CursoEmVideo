Fonte Monoespaçada

Usamos a tag code para fazer um texto.ficar.mono.espaçado porém ele perde a identação quando vc for tentar escrever linhas de comando, sendo assim usamos a tag pre. Exemplo: 


num = int(input('Digite um valor: '))
if (num % 2) == 0:
    print('O número é par')
else:
    print('É ímpar')
