Testando imagens

Uso de imagens baixadas

Para inserir uma imagem usamos a tag img, onde em src inserimos seu destino e depois há um texto alternativo (alt) que deve descrever a imagem afim de tornar a página inclusiva para PcD

Uso de imagens da internet

Também é possível carregar imagens externas. Para isso copiamos o endereço da imagem (URL).

Imagens Dinâmicas

Este tópico trata de como você deve manipular uma imagem para que ela mude seu tamanho de acordo com o tamanho do navegador utilizado. Esse tipo de imagem ajuda a evitar que o site tenha problemas de rejeição por escrolagem lateral.
Sendo assim vamos alterar o tamanho da imagem de acordo com o tamanho do navegador usando as tags picture que aparentemente encapsula configurações pra uma imagem; e a tag source:media:type que é capaz de determinar o que ocorre quando o tamanho do navegador for diferente.
Dentro da tag picture inserimos as imagens na ordem q devem aparecer de acordo com o tamanho. Um exempo é: definir a imagem grande como padrão, caso o navegador tenha 1050px (parâmetro media como max-width) usamos a imagem media (parâmetros srcset e type identificando o tipo de arquivo image/png nesse caso); e caso otamanho máximo da tela seja 750px então usamos a imagem pequena. Inserimos 50px a mais em cada um para evitar aparecer a barra de rolagem.

Inserindo Audios

Para inserir audio usamos a tag audio, como parâmetros podemos inserir controls para aparecer controles de play, pause e controle de volume; outro parâmetro é o autoplay; loop para executar a musica em looping; preload para determinar se o audio deve ser carregado antes ou depois. Para ter autoplay é interessante ter o arquivo em todos os formatos (MP3, WAV, OGG) para evitar erros de compatibilidade.

No caso de inserir os diferentes formatos, podemos usar a tag audio para encapsular as buscas dos arquivos nos diferentes formatos, e assim se um formato nao der carrega outro. Obs.: Arquivos wav sao muito pesados.

Inserindo Vídeos

Vídeos os hospedados em servidor próprio

Padrões mais utilizados e suportados são MP4, M4V WEBM e OGV. Usamos o handbreak para fazer a conversão, porém para converter para ovg pode ser usado outro sites.

Para inserir vídeos usamos a tag video e assim como nas imagens podemos usa-la para encapsular todos os tipos de video. Usamos os parâmetros controls para ter controle de play, pause e volume; usamos poster para aparecer uma imagem fixa no video; e width para determinar a largura do vídeo.

Vídeos de outros sites

Para isto usamos a tag iframe que vem em código pronto no compartilhamento do vídeo que pode ser obtido do youtube ou vimeo.
