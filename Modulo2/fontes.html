Classes e identidades (Seletores Personalizados)

Identidades são definidas a partir do parametro id e são acessadas no css por meio da hashtag #nome; elas são utilizadas como para uma única tag. Já as classes podem ser utilizadas por várias tags e são acessadas no css por meio de .nome

Pseudo Classes

Pseudo classes são usadas para mudança de estado e estão diretamente ligadas a tags já usadas. Para categorizar uma pseudoclasse usamos tag:ação

Hover
Faz algo quando o mouse esta por cima
visited
Links já visitados
Active
Quando clicado

Pseudo Elementos

Os pseudo elementos são referenciados em css por meio de duplo dois pontos (::), onde organizamos: nome::parametro.

After
É um parâmetro usado para inserir algo após o um conteúdo

Classe filha

Para associar elementos css a uma classe filha pertencente a uma classe pai utilizamos o símbolo maior que
