Modos de exibir cores em CSS

Representação de nomes

Podemos inserir uma cor ao código digitando seu nome em inglês como: blue, white, brown, etc. Observação: comandos em CSS devem ser separados por ";".

Representação por Hexadecimal

Para isso usamos # antes de inserir o código hexadecimal. Um exemplo para o azul seria: #0000ff

Representação por RGB

Usa básicamente uma tupla para armazenar 3 valores decimais de 0 a 255 para intensificar as cores: vermelho, verde e azul. Exemplo: rgb(0, 0, 255) = azul

Representação por HSL (Hue, Saturation, Luminosity)

Faz a representação da cor por matiz, saturação e luminosidade. Exemplo: hsl(240, 100%, 50%) = azul.
