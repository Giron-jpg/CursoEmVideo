Gradientes

Para inserir o Gradiente no fundo desta pagina inserimos no cabeçalho o parâmetro background-image que define uma imagem para ser inserida ao fundo. Dentro deste parâmetro inserimos a função linear-gradient (caso queira um radial use radial-gradient) que recebe como parâmetro a direção e as cores para o qual devemos fazer o gradiente. Caso o gradiente seja inserido na vertical necessitou de uma Configuaração adicional para a CSS.
Inserindo um * no estilo os parâmetros adicionados dentro das chaves servirao como comandos padrão para a CSS. Sendo assim o parametro inserido para resolver o problema do gradiente foi o height em 100%, já que ele permite extender a imagem de fundo para o tamanho total da página.
Outro problema para resolver é a borda de baixo que fica com uma linha em detrimento da imagem nao ser fixa, sendo assim insere-se também o parãmetro background-attachment como fixed.
